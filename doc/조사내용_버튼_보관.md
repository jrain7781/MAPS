# 조사내용 버튼 (보관 — 원복 후 재적용용)

카드 전체는 원복했지만, 아래 "조사내용" 버튼 디자인은 사장님 요청으로 따로 보관.
git 커밋 `7dd0fa4` 에도 전체 상태가 있음. 재적용하려면 아래를 그대로 넣으면 됨.

## 특징
- 텍스트: **조사내용** (기존 "조사내용보기"에서 축소, 오른쪽 `›` 화살표 제거)
- 박스 **높이 32px 고정** → 글자 키워도 박스 안 커짐
- 아래·오른쪽 여백 동일(12px), 우측에 붙임
- 진한 테두리(#94a3b8) + 아래 단단한 모서리(입체 눌림)

## CSS (css.html)
```css
.view-btn-header {
  display: inline-flex; align-items: center; justify-content: center; gap: 4px;
  height: 32px; padding: 0 13px; line-height: 1; overflow: hidden;
  color: #334155;
  background: linear-gradient(180deg,#f1f5f9,#dbe2ea);
  border-radius: 11px;
  font-weight: 800; font-size: 15px;   /* 필요시 조절 */
  cursor: pointer; border: 1.5px solid #94a3b8; flex-shrink: 0;
  box-shadow: 0 3px 0 #94a3b8, 0 6px 11px -3px rgba(15,23,42,.25), inset 0 1px 0 rgba(255,255,255,.9);
  transition: transform .07s ease, box-shadow .07s ease, filter .12s;
}
.view-btn-header:hover { filter: brightness(1.03); }
.view-btn-header:active { transform: translateY(3px); box-shadow: 0 1px 0 #94a3b8, 0 2px 5px -2px rgba(15,23,42,.25), inset 0 1px 0 rgba(255,255,255,.85); }
.vbh-lbl { font-weight: 800; white-space: nowrap; letter-spacing: .01em; }
```

## HTML (cItem, js-app.html) — 화살표 없이 텍스트만
```
'<button class="view-btn-header" onclick="event.stopPropagation();openImgViewer(...)" title="조사내용보기"><span class="vbh-lbl">조사내용</span></button>'
```
