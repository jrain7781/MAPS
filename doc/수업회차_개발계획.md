### ⬜ STEP 2: UI — 수업회차 좌측 영역 개편
**대상 파일**: index.html, js-app.html  
**작업 내용**:
1. `classScheduleDetailPane` 폭: `md:w-1/2` → `md:w-[35%]`  
   (좌측 리스트는 `md:w-[65%]`로 자동 확장)
2. 제목: "수업 상세 조회 항목" → "수업 회차"
3. 회차 그리드 → 배치 이전/이후 화살표 네비 + 현재 배치 회차 카드 표시
   - 배치 = class_d1_id의 타임스탬프 부분이 동일한 그룹
   - 이전 배치 / 이후 배치 화살표로 전환
4. 일반 회차 카드 형태:
   ```
   [ 1회 | 04/06 (월) | 물건: 0개 | 회원: 2명 ]  [수정] [삭제]
   ```
5. 회차 수정: 날짜 인라인 수정 (updateClassD1Date GAS 함수 추가 필요)
6. 회차 삭제: 확인 후 deleteClassD1 호출

### ⬜ STEP 3: UI — 요약 회차 카드
**대상 파일**: index.html, js-app.html  
**작업 내용**:
- 회차 목록 맨 위에 "요약" 카드 고정 표시
- 내용: 총 N회 | 총 물건 M개 | 총 회원 K명
- 요약 선택 시 하단에 전체 회원 출석표 표시
  - 행 = 회원명 (전체 회차에 등록된 유니크 회원)
  - 열 = 회차 번호 (1, 2, 3...)
  - 셀 = 참석 O / 미참석 빈값
  - 회원명 앞: 총 참석횟수
  - 회원명 마우스오버: ID, 전화, 수업명, 명의 툴팁 (updateMemberTokenInfo 방식 동일)
- GAS 추가 함수: `readAllMembersByClassId(classId)` — 해당 수업 전체 회차 회원+출석 조회

### ✅ STEP 4: 물건 탭 — 등록 기능
**대상 파일**: SheetDB.js (`addItemsToClassD1`)
**작업 내용**:
- 일괄 등록 시 items 업데이트:
  - `stu_member` = `'추천'`
  - `m_name` = `종목_수업일(yymmdd)_회차` (예: `부산단기투자클럽A_260406_3회차`)
  - `class_d1_id` = 현재 선택된 회차 ID (S열)
  - `chuchen_state` = `'신규'`
  - `chuchen_date` = 등록 시점 (yyyymmdd)
  - `bid_datetime_2` = class_d1.bid_datetime_2 → **yymmddhhmm 문자열로 변환 저장** (T열)
- 삭제 시(`removeItemFromClassD1`): `class_d1_id`, `bid_datetime_2`, `chuchen_date` 모두 초기화 → 로딩바 리셋

**상태**: ✅ 완료
