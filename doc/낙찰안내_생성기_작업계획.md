# 낙찰 안내 자동 생성기 — 작업계획

## 목표
물건 데이터 + 조사내용을 넣으면 **카페용 낙찰 안내(어그로 제목 + 카드형 본문 + 업로드용 이미지)** 를 자동 생성.

## 설계 원칙
- **숫자(수익·%)는 코드로 정확히 계산** (AI에 돈 계산 맡기지 않음 — 오류/환각 방지)
- **글(제목·긍정 문구·요약)만 AI(Claude API)가 작성**
- **전화번호는 정규식으로 자동 탐지·블러**

## 선택 방향 (2026-06-30 확정)
- 제목·문구: **AI 자동작성** (Claude API)
- 입력: **MAPS 물건에서 자동 로드**
- 출력: **HTML 미리보기 + 이미지(PNG) 자동추출**

## 아키텍처
- **위치**: MAPS(GAS) 내 신규 기능. 진입 = 입찰물건관리 물건 상세(또는 낙찰 물건)에서 `📣 낙찰 안내 생성` 버튼 → 모달.
- **입력**
  - items DB 자동: 주소 / 감정가 / 최저가 / 대지·전용 면적 / 사건번호 / 종별
  - 낙찰 결과: 낙찰가 · 입찰인원 · 차순위금액 · 매각기일 (옥션/리포트 연동 또는 입력)
  - 조사내용: 텍스트 (조사 시스템 자동로드 또는 붙여넣기)
  - 시세: 매매 / 전세 / 월세(보·월) — 조사내용에서 추출 or 입력
- **처리**
  - 수익계산(JS): 감정가 대비 %, 매매차익(시세−낙찰가), 전세 무피/플피(전세−낙찰가), 월세 수익률(실투자=낙찰가−보증금, 연임대료/실투자)
  - 전화 블러(정규식): `0\d{1,2}-\d{3,4}-\d{4}` 등 → `<span class=blur>`
  - AI(GAS UrlFetchApp → Anthropic `/v1/messages`): 입력=전화 가린 조사내용 + 숫자 요약 → 출력(JSON)= 어그로 제목 3개 + 긍정 요약 bullet + 부동산 코멘트 요약
- **출력**: HTML 카드 템플릿(`doc/낙찰안내_2024타경82847.html` 기반)에 채움 → 미리보기 → html-to-image로 PNG 추출(기존 다물건 캡처 방식 재사용) + 제목/본문 복사 버튼

## Phase
- **P1 (MVP, AI 제외)**: 모달 UI + 입력폼 + 수익계산 + 전화블러 + 템플릿 렌더 + 이미지 추출. 제목/문구는 패턴·수동.
- **P2 (AI)**: Claude API 연동 — 제목·문구 자동작성. Script Property `ANTHROPIC_API_KEY`. 모델: `claude-haiku-4-5`(저렴·빠름) 기본, 품질 필요 시 `claude-sonnet-4-6`.
- **P3 (DB 자동연동)**: 물건 선택 시 items/조사 자동로드, 낙찰결과 자동(옥션/리포트), 이미지 자동추출 마감.

## 블로커 / 확인 필요
1. **Anthropic API 키** (P2용) — Script Property에 저장. 키 발급/제공 필요.
2. **조사내용 저장 위치** — 자동로드용. 미정 시 P1은 붙여넣기로 시작.
3. **낙찰결과 소스** — 옥션 크롤링/리포트 연동 vs 수동 입력.
4. **버튼 위치 확정** — 입찰물건관리 상세 모달 권장.

## ★ 아키텍처 확정 변경 (2026-06-30) — 크롤링 매니저로
초기엔 MAPS(GAS) 내 버튼으로 계획했으나, **00.auction1 크롤링 매니저**에 넣기로 변경.

- **위치**: 00.auction1 크롤링 매니저 → **물건캡쳐 탭** 안에 **신규 "낙찰카페등록" 탭** 추가.
- **흐름**:
  1. 사용자가 **사건번호 입력 → 불러오기**
  2. 백엔드가 ① **MAPS 조사내용** 가져오기(GAS 웹 API 또는 시트 조회) ② **옥션원 사건조회 크롤링**(Selenium) → 낙찰 결과 + **사건조회 첨부 이미지 캡처**
  3. **매수인 이름 마스킹**(성만 남기고 뒤 가리기, 예 "이현민"→"이○○", 또는 블러)
  4. **카페등록 이미지 생성**: 어그로 제목 + 조사 요약 + 수익 시나리오 + **중간에 옥션 사건조회 이미지 삽입** (부동산 전화 블러)
  5. **최종 이미지 출력** — **복사하기 버튼(이미지 클립보드 복사 → 카페에 바로 붙여넣기)** + 다운로드. 제목/본문 텍스트 복사 버튼도 함께.
- **글(제목·문구)**: 무료 방침 — A.템플릿·규칙(무료) 우선, 필요 시 B.Gemini 무료 API.
- **확인 필요(빌드 시)**: 00.auction1의 물건캡쳐 탭 구조 / MAPS 조사내용 접근용 GAS 엔드포인트 / 옥션원 사건조회+낙찰결과 크롤링(이미 project_auction1_search_mechanism 매각 크롤러 존재) / 이미지 합성(Python PIL 또는 html→image).
