# [MAPS] 낙찰 카페글 생성 (어그로·고급)

## 목적
회원/비회원이 보고 **"나도 경매하고 싶다 · 이렇게 버는구나 · 부럽다"** 를 느끼게 하는 후킹 콘텐츠. 우리 회원의 낙찰 성공을 **증빙과 함께 자랑스럽고 감성적으로**, 그러나 **허위·과장 없이**.

## ★ 절대 원칙 (지난 실패에서)
1. **조사내용 원문을 카드에 그대로 넣지 말 것.** "경기도… 인구… 공급… 부동산 A/B/C 나열" 같은 raw 덤프는 금지. **혹하게 유리한 정보만 가공·요약**해서 `ncSummary`에 넣는다. (데이터를 **가공**하는 게 내 일)
2. **친근한 인사**: 카드 최상단은 딱딱한 "○○등급 회원님"이 아니라 **이름(성 제외) + 회원님** — 예 "현민 회원님". (카드 코드가 등급/이름을 헤더에 자동 표시하므로, 제목은 어그로 문구로.)
3. **감성·짜릿 톤**: "차순위와 단 230만원 차이 — 짜릿한 역전 낙찰!", "5명 뚫고", "부럽다면" 같은 후킹. 밋밋한 사무적 나열 금지. (카드가 차순위/경쟁 문구는 자동으로 넣어줌 → 제목·요약을 거기에 맞춰 감성적으로.)
4. **증빙 이미지는 카드에 자동 삽입됨**(옥션 매각결과 캡처). 크롤 후 `ncCrawl.screenshot_path`가 있으면 카드 상단 "📸 실제 낙찰 증빙"으로 자동 렌더. 별도 작업 불필요(누락됐으면 [▶실행]이 안 된 것).

## 1) 데이터 읽기 (크롤링 매니저)
Chrome MCP로 `http://localhost:8765` → 물건캡쳐 ▸ 낙찰카페등록 탭. 사용자가 [📥불러오기]+[▶실행]을 이미 눌러 연동된 상태여야 함(안 됐으면 먼저 실행 안내).
- 연동 데이터(폼 input value): `ncMember`, `ncGrade`(MAPS), `ncBuyer`, `ncDate`, `ncAppr`, `ncMin`, `ncBid`, `ncGongsi`, `ncSecond`, `ncCnt`, `ncAddr`, `ncSale`/`ncJeonse`/`ncWolBo`/`ncWol`, `ncHoga`, `ncJosa`(조사내용 전문 — 읽기용, 카드엔 원문 금지)
- **내가 가공해 채울 것**(감정가·최저가·낙찰가·공시지가는 옥션/MAPS 자동, 손대지 말 것):
  - **시세 `ncSale`/`ncJeonse`/`ncWolBo`/`ncWol`** ← `ncJosa`의 **부동산(■) 섹션들**에서 2~3곳 집계. 대표값=중앙/평균(범위면 보수적 하한), 억/만→숫자. 값 없으면 비워둠(지어내지 말 것).
  - **호가 `ncHoga`** ← `ncJosa` 상단 "N. 호가" 라인(집주인 부르는값). ★부동산 시세와 별개.
  - **조사 요약 `ncSummary`** ← `ncJosa`에서 **유리한 포인트만 3~5줄**, **한 줄=한 항목**(카드가 ✓ 불릿으로 렌더). 입지·역세권·전월세 수요·실거래 이력·개발호재 등. 부정/리스크는 넣지 않거나 매우 절제. **전화번호·연락처 금지**(카드가 블러도 하지만 애초에 넣지 말 것).
- **결과 주입**: 어그로 제목 → `ncTitle`. 조사 요약 → `ncSummary`. 시세/호가 → 각 칸. 채우면 카드 자동 재생성(입력 이벤트).

## 2) 숫자 계산 (연동값 기준 — 지어내지 말 것)
- 낙찰률 = 낙찰가 ÷ 감정가
- 매매차익 = 매매시세 − 낙찰가 / 전세 무피·플피 = 전세가 − 낙찰가 / 월세수익률 = 월세×12 ÷ (낙찰가 − 월세보증) ×100
- 모든 시세·수익은 **추정 + 세금·수리·명도비 별도** 명시.

## 3) 제목 (★회원 등급 반드시 언급, 후보 3개)
- 구성: **회원등급** + 핵심 숫자(감정가 대비 %·차익·무피) + 감탄/후킹
- 예: "[프리미엄PT·실버] 현민 회원님, 감정가 42%에 낙찰 — 전세만 놔도 +4,200만!" / "5명 뚫고 일산 아파트 9,800만원 낙찰 (실버 회원 실화)"
- 금지: 매수인 실명, 전화번호, "확정수익/보장", 허위.

## 4) 본문(=카드) 구성 — 카드가 렌더하는 순서
1. 헤더: 🏆 **이름 회원님 · 등급** 낙찰 성공(자동) + 어그로 제목(`ncTitle`)
2. 📸 낙찰 증빙 캡처(자동)
3. 📌 낙찰 요약: 감정가·최저가·낙찰가(%)·입찰 N명·매수인(마스킹) + 차순위/경쟁 짜릿 문구(자동)
4. 💰 수익 시나리오: 매매차익/전세무피/월세수익률(시세 채우면 자동)
5. 🔎 현장 조사 요약: `ncSummary`(내가 가공한 유리한 포인트만)
- **채팅 출력**: 제목 후보 3개 + (요약/시세 주입 완료 보고). 사용자 선택 시 `ncTitle` 교체.

## 필수 규칙
- 매수인 이름 **성만 남기고 마스킹**(카드 자동). 연락처·주민번호 노출 금지.
- 조사내용 원문/전화번호 카드 금지 — 가공 요약만.
- 수익·시세 **추정 + 비용 별도** 명시. 확정수익·허위·과장 금지.
- 숫자는 매니저 연동값(코드계산) 사용. 없는 값 지어내지 않음.
