# 카카오톡 매크로 서버 설정 방법 (README)

다른 PC 또는 자리에서 MAPS 환경과 연동하여 해당 카카오톡 자동화 발송 기능을 사용하시려면, 아래 단계별 절차에 따라 필수 환경을 구성해주셔야 합니다.

## 1. 사전 요구사항 확인
- **운영체제**: Windows 환경을 권장합니다.
- **카카오톡 실행**: 대상 PC에 카카오톡이 설치되어 있어야 하며 작동할 카카오톡 계정으로 로그인되어 있어야 합니다.

## 2. Python 설치
1. [Python 공식 홈페이지](https://www.python.org/downloads/)에 접속합니다.
2. 최신 버전의 Python(Windows Installer)을 다운로드하고 설치합니다.
3. **[중요]** 설치 화면 하단의 **`Add Python to PATH`** 체크 박스를 반드시 체크한 후 설치를 진행하세요.

## 3. 필수 패키지 설치
1. 키보드의 `Windows 키 + R`을 누르고 `cmd`를 입력해 명령 프롬프트를 엽니다 (또는 PowerShell, VSCode 터미널 등).
2. 아래 명령어를 입력하여 로컬 서버 구동과 클립보드 제어에 필요한 라이브러리를 설치합니다.
   ```bash
   pip install pyautogui pyperclip pygetwindow
   ```

## 4. 서버 통신 파일 (kakao_server.py) 세팅
1. 본 폴더에 있는 `kakao_server.py` 파일을 카카오톡이 설치된 PC의 원하는 경로에 복사합니다.
2. 명령 프롬프트나 VSCode 터미널에서 해당 파일이 있는 경로로 이동합니다 (예: `cd C:\LJW\01. SYSTEM\MAPS_TEST\01. kakao`).
3. 아래 명령어를 실행하여 서버를 활성화합니다.
   ```bash
   python kakao_server.py
   ```
4. 터미널 창에 **"카카오톡 매크로 서버가 8000 포트에서 시작되었습니다."** 라고 표시되면 세팅 완료입니다.
   - **(주의!) 사용하는 동안 이 터미널 창은 절대 끄지 마시고 켜두어야 합니다.**

## 5. MAPS 웹에서 테스트하기
1. MAPS 웹 페이지(localhost 또는 서버)에 접속합니다.
2. 입찰 모달이나 회원 상세창에서 [카톡 전송] 버튼을 누릅니다.
3. PC 화면에서 카카오톡 창이 최상단 활성화(또는 선택) 된 상태를 유지하시면, 지정된 이름으로 검색하여 자동으로 메시지를 발송합니다.

## ※ 트러블슈팅 및 주의사항
- 카카오 전송 클릭 시 팝업에 '로컬 통신 실패' 메시지가 뜬다면, 서버(`kakao_server.py`)가 켜져 있는지 확인하세요.
- 매크로 오작동을 피하기 위해 카카오톡 메인창이 화면에 확실히 띄워져 있는 상태에서 사용을 진행하는 것을 권장합니다.
- `환경설정 > 카카오전송` 탭에서 **자동 전송(엔터)** 여부를 입맛에 맞게 ON/OFF 할 수 있습니다. 처음 사용 시 반드시 자동으로 보내지 않도록 끄고 작동 테스트를 해보시는 것을 권장합니다.
