# MJ 크롤링 매니저 Chrome 확장 - 설치 가이드

Chrome 툴바 아이콘 한 번 클릭으로 크롤러 서버 실행 + 새 탭 열기.

> ⚠️ Windows cmd 인코딩 이슈로 `register.bat` 안내문은 영문으로 표시됩니다.
> 본 가이드의 한글 설명을 보면서 진행하세요.

## 1단계 — Chrome 에 확장 로드

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우상단 **"개발자 모드"** ON
3. **"압축해제된 확장 프로그램 로드"** 클릭
4. 폴더 선택:
   ```
   C:\LJW\01. SYSTEM\MAPS_TEST\00.auction1\chrome_ext
   ```
5. "MJ 크롤링 매니저 실행기" 카드 로드 확인
6. **카드 ID 복사** (32자 영문, "ID: " 옆에 표시)

## 2단계 — Native Messaging 호스트 등록

1. 파일 탐색기에서:
   ```
   C:\LJW\01. SYSTEM\MAPS_TEST\00.auction1\native_host
   ```
2. `register.bat` **더블클릭** (영문 콘솔창 뜸)
3. 메시지 "Paste extension ID and press Enter:" 가 보이면
4. **확장 ID 붙여넣기** → Enter
5. "[OK] Registry registered." 보이면 성공 → 아무 키 → 닫기

## 3단계 — 사용

1. **Chrome 완전 종료 후 다시 실행** (중요 — 안 하면 native host 인식 안됨)
2. 툴바 우측 퍼즐 아이콘 → 핀으로 MJ 확장 고정 (선택)
3. **MJ 아이콘 클릭** → 자동으로:
   - 서버 안 떠있으면 `crawler.py` 백그라운드 실행
   - 떠있으면 그대로 사용
   - 새 탭에 `http://localhost:8765/` 열림

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| 아이콘 눌러도 반응 없음 | Chrome 완전 종료 후 재시작 |
| "네이티브 호스트 호출 실패" 새 탭 | `register.bat` 다시 실행 (확장 ID 재등록) |
| `python --version` 안 됨 | Python 설치 + PATH 등록 |
| 포트 8765 충돌 | `netstat -ano \| findstr :8765` → 해당 PID 종료 |
| 확장 ID 가 바뀜 | 확장 제거 후 재로드 → 새 ID 로 `register.bat` 재실행 |

## 제거

`native_host\unregister.bat` 실행 → `chrome://extensions` 에서 확장도 제거.
